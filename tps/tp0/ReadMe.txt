Program that compresses and decompresses text, using Lempel-Ziv-Welch compression/decompression algorithm.

Usage:
	./tp0 -i entrada.txt -o salida.txt -p compress
	or
	./tp0 -i entrada.txt -o salida.txt -p decompress

	Default uses std in, std out and compress. 
	Long optiones: --input, --output, --process

Test cases can be found in the test directory.
